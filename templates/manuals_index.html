How to save a manual as PDF

  1. Click a manual above. It opens in a new tab.
  2. The browser's print dialog opens automatically (or press CtrlP).
  3. Under Destination pick Save as PDF.
  4. Set Layout to Portrait, Paper size to A4.
  5. Click Save. The PDF captures the current version of the app — re-save after upgrades to stay in sync.
